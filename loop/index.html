Loop ⟩

Introduction

Loop is a little functional language that runs on the JVM.  It takes advanced functional features and makes them easy and fun to use. Loop integrates tightly with Java so using existing code or libraries is a cinch.

Ironically, there is no loop construct in Loop.

Examples

  • Hello World
      main ->
    print('Hiya Loopy!')
  • String Interpolation
      greet(name) ->
    print("Greetings, @{name}!")
  • List Comprehensions
      # returns ['HUGH', 'FRY']
  main ->
    (name.toUpperCase() for name in ['hugh', 'fry'])
  • Pattern matching 1
      factorial(i) =>
    0        : 1
    *        : i * factorial(i - 1)
  • Pattern matching 2
      # returns 'olleh' for 'hello'
  reverse(string) =>
    ''        : ''
    (c:rest)  : reverse(rest) + c
  • Guards
      # regex matching with guards
  show(name) =>
    /Stephen.*/  | name.endsWith('Fry')     : 'QI'
                 | name.endsWith('Colbert') : 'The Report'
                 | else                     : 'boring'
  • Where Scope
      # prints hours in 3 years: 26208
  main ->
    print(3 * year)
    where
      day: 24
      week: 7 * day
      year: 52 * week
  • Objects
      # A type declaration
  class Star ->
    String name
    Double mass

  main ->
    new Star(name: 'Proxima Centauri',
             mass: 0.123)
  • Modules
      require loop.io

  main ->
    io.read('story.txt')
  • Java Interop
      # prints current time
  main ->
    `java.lang.System`.currentTimeMillis()
