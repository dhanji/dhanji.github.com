Loop ⟩

Introduction

Loop is a little functional language that runs on the JVM.  It takes advanced functional features and makes them easy and practical. Loop integrates tightly with Java so you can use existing libraries and code easily.

Ironically, there is no loop construct in Loop.

Features

  • List Comprehensions
      # returns ['HUGH', 'FRY']
  main ->
    (name.toUpperCase() for name in ['hugh', 'fry'])
  • Pattern matching
      # returns 'olleh' for 'hello'
  reverse(string) =>
    ''        : ''
    (c:rest)  : reverse(rest) + c
  • Where Scope
      # prints hours in 3 years: 26208
  main ->
    print(3 * year)
    where
      day: 24
      week: 7 * day
      year: 52 * week
  • Guards
      # TV hosts by regex, and their shows
  show(name) =>
    /Stephen.*/  | name.endsWith('Fry')     : 'QI'
                 | name.endsWith('Colbert') : 'The Report'
                 | else                     : 'boring'
  • Objects
      # A type declaration
  class Star ->
    String name
    Integer mass
    Tensor gravity
    Boolean blackHole
  • Modules
  • Java Interop
      # prints current time
  main ->
    `java.lang.System`.currentTimeMillis()

Code Example

Here's the common factorial function in Loop using pattern matching: 

  factorial(i) =>
    0        : 1
    *        : i * factorial(i - 1)
